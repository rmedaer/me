# Me, myself and I

A latex exercise: my resume !

## Build

Please install following dependencies:

  * texlive-xetex
  * texlive-bibtex-extra
  * Ubuntu fonts

I might missed some dependencies, feel free to complete through PR ;-)

Then run: 

```bash
make
```

That's it !
